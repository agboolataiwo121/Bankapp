# Bankapp





\# Taiwo Bank App



A Python console-based banking application with MySQL database.



\## Features

\- User registration and email verification

\- Secure login with OTP-based 2FA

\- Deposit, withdrawal, and transfer

\- Transaction history

\- Admin dashboard

\- Email alerts for all transactions

\- Password reset via OTP



\## Requirements

\- Python 3.8+

\- MySQL / MariaDB

\- XAMPP or any MySQL server



\## Installation



1\. Clone the repository:

&#x20;  git clone https://github.com/agboolataiwo121/Bankapp.git



2\. Install dependencies:

&#x20;  pip install mysql-connector-python python-dotenv



3\. Start your MySQL server (XAMPP)



4\. Configure SMTP credentials in bankconfig.py



5\. Run the app:

&#x20;  python bankapp.py



\## Usage

\- Register a new account

\- Verify your email with the OTP sent to your inbox

\- Login and access your dashboard

\- Admin access is available via the role column in the database
